# shellgen
**shellgen** is a custom script that prints in prompt a lot of different reverse shells options with a given IP and PORT to copy and paste

# Supported
- Linux
- Windows PowerShell
- PHP

# Usage

### revshell
```
[!] usage: python revshell.py <IP> <PORT>
```
### webshell
```
[!] usage:
    python webshell.py
    python webshell.py <METHOD> <PASSWD>
```
